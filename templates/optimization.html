🎯 Что такое оптимизация?

Оптимизация — это процесс нахождения наилучшего решения среди всех возможных. В математике и программировании оптимизация используется для поиска минимума или максимума функции.

Найти min/max f(x) при ограничениях g(x) ≤ 0

Применяется в экономике, инженерии, машинном обучении и многих других областях.

📈 Метод золотого сечения

Один из простейших методов одномерной оптимизации для нахождения минимума функции на заданном интервале.

📝 Алгоритм: 
1. Задаем интервал [a, b] и точность ε
2. Вычисляем точки:
   x1 = b - (b-a)/φ
   x2 = a + (b-a)/φ
   где φ = (1+√5)/2 ≈ 1.618 (золотое сечение)
3. Если f(x1) < f(x2), то новый интервал [a, x2]
   Иначе [x1, b]
4. Повторяем пока |b-a| > ε

🚀 Градиентный спуск

Основной метод оптимизации в машинном обучении. Итеративно движется в направлении антиградиента для поиска минимума.

xk+1 = xk - α·∇f(xk)

Где α — скорость обучения (learning rate).
